# Lisp Functions

The `kakei_lisp` dialect provides a set of built-in functions for transforming and analyzing transaction data. This reference covers all available functions.

- [Core Functions](./lisp-functions/core-functions.md) - lambda, define, if
- [List Operations](./lisp-functions/list-operations.md) - cons, car, cdr
- [Comparison Functions](./lisp-functions/comparison-functions.md) - equal?, null?
- [Association List Functions](./lisp-functions/association-list-functions.md) - assoc and working with alists
- [Table Manipulation](./lisp-functions/table-manipulation.md) - group-by and other table operations
