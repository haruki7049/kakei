# How to Build Locally

To build kakei locally for development:

## Clone the Repository

```bash
git clone https://github.com/haruki7049/kakei.git
cd kakei
```

## Build with Cargo

```bash
cargo build
```

## Run Tests

```bash
cargo test --workspace
```

## Run the Application

```bash
cargo run -- --help
```

For more details, see the [Development](../development.md) guide.
